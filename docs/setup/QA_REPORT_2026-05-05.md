# KCG QA Report - 2026-05-05

## Design Thesis

KCG should feel like a Jongno gold-exchange price desk: posted prices and product categories first, operational copy second, and admin screens that behave like a control console rather than a marketing page.

## QA Scope

- Public routes: `/`, `/prices`, `/products`, `/services`, `/company`, `/about`, `/announcements`.
- Admin routes: `/admin`, `/admin/prices`, `/admin/products`, `/admin/announcements`, `/admin/launch`.
- Viewports: mobile 390px and desktop 1440px.
- Evidence: Playwright tests, rendered/source audit, screenshot set, and manual screenshot review.

## Issues Found And Fixed In This Pass

- Mobile `/products` first viewport used too much vertical space before the catalog controls. Fixed by tightening the hero height, top padding, catalog spacing, and sort-control rhythm.
- `/admin/prices` automatic-price switch could show a saved OFF state while the screen preview was ON, which read as contradictory. Fixed by separating `저장된 운영 상태` from `화면 상태`.
- Admin screenshot evidence did not include `/admin` and `/admin/products`. Fixed by adding opt-in admin dashboard and products screenshots to the screenshot helper.
- A regression test now checks that mobile `/products` reaches the catalog controls in the first viewport.

## Remaining Improvements

- Replace generated product images with real KCG-owned photos for gold bars, silver bars, pure-gold gifts, jewelry buying, and B2B packaging.
- Confirm real operating product list, weights, 공임, margin, and whether each category should show 참고가 or 문의.
- Rotate production admin password before search launch.
- Decide whether automatic price checks need Vercel Pro/external scheduler beyond Hobby-plan daily cron.
- Keep robots/noindex blocked until explicit public-launch approval.
