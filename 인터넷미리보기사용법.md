# 인터넷 미리보기 배포 사용법

컨펌용 인터넷 링크를 다시 만들 때는 아래 순서로 진행합니다.

## 1. Vercel 로그인

프로젝트 폴더에서 [1-vercel-login.cmd](C:\Users\i\Desktop\codex\사이트시안\1-vercel-login.cmd) 를 실행합니다.

- 브라우저 승인 페이지가 뜨면 Vercel 로그인을 승인합니다.
- 승인이 끝나면 이 창은 닫아도 됩니다.

## 2. 미리보기 배포

[2-preview-deploy.cmd](C:\Users\i\Desktop\codex\사이트시안\2-preview-deploy.cmd) 를 실행합니다.

- 컨펌용 프로젝트 이름은 `kcg-confirm-preview` 입니다.
- 배포가 끝나면 인터넷 미리보기 URL이 화면에 표시됩니다.
- 관리자 로그인 주소와 비밀번호도 함께 표시됩니다.
- 이 스크립트는 프로젝트 내부 캐시를 사용해서 npm/Vercel 권한 문제를 피하도록 설정되어 있습니다.

## 3. 배포 전후 검증

[3-validation-check.cmd](C:\Users\i\Desktop\codex\사이트시안\3-validation-check.cmd) 를 실행합니다.

- `lint`, `typecheck`, `build` 를 순서대로 확인합니다.
- Playwright로 홈과 시세 페이지를 열고 스크린샷을 저장합니다.
- 스크린샷은 프로젝트 안 `.playwright-cli` 폴더에 저장됩니다.

## 4. 배포 후 확인할 주소

- 홈: `배포URL`
- 시세: `배포URL/prices`
- 공지사항: `배포URL/announcements`
- 상품·서비스: `배포URL/services`
- 거래 안내: `배포URL/about`
- 관리자 로그인: `배포URL/admin/login`

## 현재 배포 기본값

- 관리자 비밀번호: `0000`
- 검색엔진 노출: preview 에서는 자동 차단
- 데이터 모드: mock
- 실제 연락처: `02-747-1802`, `서울 종로구 돈화문로6가길 12 골든타워 303호`

## 참고

- preview 배포는 컨펌용입니다.
- 최종 공개 전에는 사업자 정보, 강한 관리자 비밀번호, 실제 도메인 연결, 검색 허용 설정을 확인해야 합니다.
